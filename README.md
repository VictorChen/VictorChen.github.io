# Victor Chen
Website built on [React](https://reactjs.org/) + [Next.js 15](https://nextjs.org/) + [Tailwind CSS v4](https://tailwindcss.com/)

## Local Development

```
npm install
npm run dev
```

Push changes to `dev` branch.

## Test Production Version

```
npm run build
npm start
```

## Deploy to Github Pages

```
npm run deploy
```
